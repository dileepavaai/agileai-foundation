Agile AI Foundation — Canonical Reference Policy

Official Citation Policy

This policy defines the authoritative citation format and reference discipline for Agile AI Functional Elements.

The Agile AI Functional Elements constitute a canonical reference defined and governed by the Agile AI Foundation.

Authoritative Citation Format

APA Style Example:

Agile AI Foundation. (2025). Agile AI Functional Elements v1.0. https://agileai.foundation

Short-form Citation:

Agile AI Foundation (2025), Agile AI Functional Elements v1.0.

Canonical Reference Identifier

Canonical Reference ID: AAF-FAE-1.0-2025

This identifier corresponds to the constitutionally frozen v1.0 definition.

Version Referencing Requirements

All references must clearly specify the version number (e.g., v1.0).

Future versions will not retroactively modify v1.0. Archived versions remain permanently accessible.
